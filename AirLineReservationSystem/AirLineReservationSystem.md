# ykProjects

A small collection of Python projects by Y‑K‑SAI‑SRIKAR — currently featuring an Airline Reservation System.

## 🔹 Projects

### Airline Reservation System  
- Python script: `AirLineReservationSystem.py`  
- Data files: `flights.txt`, `bookings.txt`  
- A basic console‑based application that simulates flight booking / reservation logic.  

## 📁 Repository Structure

ykprojects/
├── AirLineReservationSystem.py  
├── flights.txt         # Flight data  
└── bookings.txt        # Booking records  

## 🚀 Getting Started

1. Clone the repository  
   git clone https://github.com/Y-K-SAI-SRIKAR/ykProjects.git

2. Navigate to the project folder  
   cd ykprojects

3. Run the application using Python  
   python AirLineReservationSystem.py

## ✅ Usage & Features
- Browse available flights  
- Book a seat  
- Store bookings persistently  
- Load flight data from `flights.txt`
